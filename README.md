# 🚀 نظام أمان الطبي المتكامل - الإصدار الهجين (Aman Medical CMS)

نظام متطور وشامل لإدارة المنشآت الطبية يعمل بنظام **المزامنة الهجينة (Hybrid Sync)** لضمان العمل تحت أي ظرف (متصل/غير متصل).

## 🌟 الميزات الفريدة
*   **ثنائية التخزين:** دمج Firebase (للمزامنة السحابية) مع Dexie/IndexedDB (للسرعة المحلية).
*   **دعم العمل بدون إنترنت:** استمر في إدارة المرضى والعيادات حتى عند انقطاع الشبكة.
*   **المزامنة الفورية (Real-time):** التغييرات تظهر في جميع الأجهزة فور حدوثها.
*   **بوابة المريض الذكية:** تطبيق متكامل للمرضى لمتابعة سجلاتهم ومواعيدهم.
*   **الأتمتة الكاملة (DevOps):** بناء ونشر تلقائي عبر GitHub Actions.

## 🏥 الخدمات والموديولات المشمولة
1.  **لوحة التحكم (Dashboard):** عرض إحصائيات حقيقية لتدفق المرضى والإيرادات.
2.  **السجل الطبي (EMR):** إدارة كاملة لسجلات المريض، التاريخ الطبي، والعلامات الحيوية.
3.  **العيادة والفرز (OPD):** معاينة طبية متكاملة مع دعم التشخيصات والوصفات.
4.  **المختبر والأشعة:** إدارة الطلبات والنتائج رقمياً بشكل كامل.
5.  **الصيدلية والمخزون:** تتبع دقيق للأدوية والمستلزمات مع تنبيهات النقص.
6.  **الأقسام والتنويم (IPD):** إدارة الأسرة، الغرف، وعمليات دخول وخروج المرضى.
7.  **الطوارئ (ER):** نظام فرز طبي سريع (Triage) للحالات العاجلة.
8.  **المالية والمحاسبة:** إصدار فواتير ذكية، تتبع إيرادات، وتصدير تقارير مالية.
9.  **الموارد البشرية (HR):** إدارة شؤون الموظفين، الأطباء، وصلاحيات النظام.

## 🤖 الأتمتة والنشر (GitHub Automation)
لقد تم تجهيز النظام بنظام CI/CD احترافي موجود في `.github/workflows`:
*   **android-build.yml:** يقوم تلقائياً بـ:
    1. بناء نسخة الويب (Vite).
    2. فحص جودة الكود (Linting).
    3. النشر لـ Firebase Hosting.
    4. بناء تطبيق Android (APK) ورفعه كـ Artifact قابل للتحميل.

### 🔑 إعدادات GitHub المطلوبة
لتفعيل النشر التلقائي، قم بإضافة الـ Secrets التالية في مستودعك بـ GitHub:
*   `FIREBASE_SERVICE_ACCOUNT_KEY`: مفتاح حساب الخدمة من Google Cloud.
*   `GEMINI_API_KEY`: مفتاح الذكاء الاصطناعي (اختياري).
*   `FIREBASE_PROJECT_ID`: معرف المشروع في Firebase.

## 📱 تطبيقات الجوال
تم دمج **Capacitor** لتحويل الويب إلى تطبيق Android حقيقي:
*   مزامنة ملفات الأندرويد: `npm run android:sync`
*   فتح المشروع في Android Studio: `npm run android:open`

## 📋 إرشادات البدء السريع
1.  تثبيت المكتبات: `npm install`
2.  بدء التطوير: `npm run dev`
3.  بناء الإنتاج: `npm run build`
